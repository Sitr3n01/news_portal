1Publicar um artigo

1.1 Criar um novo artigo

1

Na barra lateral, clique em Blog da Kelly → Artigos → Adicionar Artigo.

Campos obrigatórios

CampoDescrição
TítuloCabeçalho principal da matéria (H1 na página)
URL amigável (slug)Gerado automaticamente do título. Ex.: minha-materia → kellyfarias.com.br/news/minha-materia/
CategoriaSeção principal (aparece no menu de navegação)
AutorUsuário responsável pela matéria

Campos opcionais

CampoDescrição
ResumoTexto curto exibido nas listagens e compartilhamentos
Imagem de capaRedimensionada e otimizada automaticamente (JPEG, máx. 1200×800px)
Legenda da imagemTexto exibido abaixo da imagem de capa
TagsPalavras-chave livres. Digite para buscar ou criar.
DestaqueMarque para posição de destaque na home do portal
Título SEOComo aparece no Google (máx. 70 caracteres). Se vazio, usa o título.
Descrição SEODescrição para buscadores (máx. 160 caracteres). Se vazia, usa o resumo.

Status do artigo

StatusEfeito
RascunhoSalvo mas invisível no site público
PublicadoVisível no site. A data de publicação é marcada automaticamente na primeira vez que você salva com este status.
ArquivadoRemove o artigo do site sem excluir do banco

1.2 Blocos de conteúdo

O corpo do artigo é composto por blocos ordenados, adicionados na seção "Blocos de conteúdo" dentro do formulário.

Tipo de blocoPara que serve
TextoParágrafos, subtítulos, listas, links, negrito/itálico.
ImagemImagem da Biblioteca de Mídia, com legenda opcional.
Vídeo / PostLink de YouTube, Instagram ou TikTok. O embed é gerado automaticamente.
Use o campo Ordem para controlar a sequência. O campo Conteúdo do artigo é gerado automaticamente a partir dos blocos.
O sistema sanitiza o HTML ao salvar e monta embeds a partir da URL, sem colar HTML cru no artigo.

2Categorias e tags

Categorias

Organizam o portal em seções. Categorias-raiz (sem categoria pai) aparecem como abas no menu de navegação.

  1. Clique em Blog da Kelly → Categorias → Adicionar Categoria.
  2. Preencha: Nome, URL amigável, Descrição (opcional), Categoria pai (opcional, para subcategorias) e Ordem.
  3. Salve.

Tags

Palavras-chave livres que criam páginas de arquivo automáticas para leitores navegarem.

  1. Clique em Blog da Kelly → Tags → Adicionar Tag.
  2. Preencha Nome e URL amigável.
  3. Para usar num artigo: no campo Tags do formulário do artigo, pesquise pela tag ou crie ali mesmo.

3Newsletter

Como funciona — duas etapas separadas

1. Publicar artigo
→

→
2. Disparar manualmente
→
E-mails saem para os inscritos
Publicar um artigo não envia o e-mail automaticamente. É necessário disparar manualmente pela ação do admin (ou um administrador técnico executa o comando de envio).

3.1 Disparar a newsletter pelo admin

  1. Clique em Blog da Kelly → Artigos.
  2. Marque os artigos publicados que deseja enviar.
  3. No menu Ação, selecione "Enviar Newsletter para inscritos" e clique em Executar.
  4. Um resumo de resultado (enviadas / falhas / ignoradas) aparecerá na tela.
Esta ação pode ser executada mesmo em artigos já enviados anteriormente, permitindo reenvio pontual.

3.2 Monitorar entregas

Acesse Blog da Kelly → Entregas de Newsletter. Cada linha representa um par (artigo × inscrito):

CampoSignificado
StatusPendente (aguardando), Enviado, Falhou, Ignorado (inscrito inativo)
TentativasQuantas vezes o sistema tentou enviar
Último erroMensagem de erro detalhada, se houver falha
Enviado emHorário de confirmação do envio bem-sucedido

3.3 Gerenciar inscritos

  1. Clique em Blog da Kelly → Newsletter.
  2. Para cancelar manualmente: abra o registro e desmarque Ativo.

3.4 Pré-visualizar o e-mail

Para ver como o e-mail ficará antes de enviar, acesse no navegador (requer login de staff):

seudominio.com.br/news/newsletter/preview/<id-do-artigo>/

3.5 Configurar o remetente

Se os e-mails saem com remetente genérico:

  1. Clique em Sistema → Configurações do Site.
  2. Preencha E-mail remetente da Newsletter (ex.: noticias@kellyfarias.com.br) e Nome remetente da Newsletter (ex.: "Blog da Kelly").
  3. Salve.

4Comentários

Leitores registrados podem comentar nos artigos.

Moderar comentários

  1. Clique em Blog da Kelly → Comentários.
  2. Para ocultar sem excluir: abra o registro, desmarque Visível e salve. O comentário desaparece do site mas permanece no banco.
  3. Para excluir permanentemente: clique em Excluir na tela do comentário.
O autor do comentário pode excluir o próprio comentário diretamente na página do artigo no site. Comentários de outros usuários só são removidos por administradores no painel.

5Biblioteca de mídia

Repositório central de arquivos compartilhado entre os dois portais.

Enviar um arquivo

  1. Clique em Sistema → Biblioteca de Mídia → Adicionar Arquivo de Mídia.
  2. Preencha: Título (nome descritivo), Arquivo (upload do computador), Tipo de arquivo (Imagem/Documento/Vídeo/Áudio/Outro), Texto alternativo (para imagens — acessibilidade e SEO) e Pasta (opcional).
  3. Salve.

Organizar em pastas

  1. Clique em Pastas de Mídia → Adicionar, dê um nome e, opcionalmente, selecione uma Pasta superior para criar subpastas.
  2. Ao enviar arquivos, selecione a pasta no campo Pasta.

6Usuários e acesso

Esta seção é para Super Administradores apenas.

Criar um usuário

  1. Clique em Sistema → Usuários → Adicionar Usuário.
  2. Preencha Username, E-mail, Senha e escolha o Cargo:
CargoAcesso
Super AdministradorAcesso total ao sistema
Administrador KomunikiHome, páginas, depoimentos, mensagens, mídia
Editor de NotíciasArtigos, categorias, tags, comentários, newsletter, mídia
O sistema sincroniza automaticamente o grupo de permissões com o cargo selecionado. Mudar o cargo revoga os privilégios do cargo anterior.

Alterar a senha

  1. Abra o usuário em Sistema → Usuários.
  2. Clique no link "Alterar senha" no topo do formulário.
  3. Digite a nova senha duas vezes e confirme.

7Perguntas frequentes

O artigo não aparece no site

Verifique: (1) o status está como Publicado? (2) o site correto está selecionado no campo Site do artigo?

A newsletter não foi enviada mesmo publicando o artigo

Esperado por design. Publicar não envia automaticamente. Use a ação manual descrita na seção 3.1.

Um inscrito recebeu o e-mail com links quebrados

Verifique em Sistema → Sites se o campo Domain está correto (ex.: kellyfarias.com.br). Links do e-mail usam este domínio.

Como ver quantas pessoas acessaram o artigo?

O campo Visualizações é contado automaticamente por sessão (um acesso por sessão de navegador). Veja na lista de artigos ou no detalhe do artigo.

O artigo continua listado como "pendente de newsletter"

Há uma ou mais entregas com status Falhou em aberto. Acesse Entregas de Newsletter, verifique o campo Último erro e tente reprocessar via ação manual ou solicite ao administrador técnico.
