Como acessar o painel

Acesse seudominio.com.br/admin/ com seu usuário e senha. A tela inicial exibe um painel de métricas resumidas. A barra lateral esquerda contém todos os menus organizados por contexto.

Quem pode fazer o quê

O sistema usa cargos para controlar o que cada pessoa pode ver e editar:

Cargo Acesso permitido
Super Administrador Tudo no sistema, incluindo usuários, configurações, vagas e relatórios completos
Administrador Komuniki Home da escola, páginas, diferenciais, depoimentos, mensagens de contato, biblioteca de mídia
Editor de Notícias Artigos, categorias, tags, comentários, inscritos na newsletter, biblioteca de mídia
Regra de ouro:

Toda alteração feita no painel fica visível no site público imediatamente após clicar em Salvar. Não existe botão de "publicar servidor".
